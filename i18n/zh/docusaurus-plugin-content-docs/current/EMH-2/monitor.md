# 系统监控与实时概览

## 1. 功能概述

系统首页（实时概览）用于展示家庭能源系统的整体运行状态，包括光伏发电、储能、电网及负载等关键环节的运行情况，并提供能量流向及经济收益的可视化信息。

通过该页面，用户可快速了解系统当前运行状态及能量利用效率。

<img src={require("./img/system_overview.png").default} width="960" /> 


## 2. 页面说明

首页主要由以下几个区域组成：

### 2.1 系统拓扑展示（核心区域）

页面中央以图形化方式展示家庭能源系统结构，包括：
- 光伏发电系统
- 储能系统（电池）
- 家庭负载
- 电网连接
- 电动车（如有）


### 2.2 时间维度切换

页面右侧及下方提供时间维度选择：
- 今日
- 昨日
- 本周 / 上周

👉 用于切换不同时间范围的数据展示

### 2.3 成本与收益分析

右侧展示能源经济数据，包括：
- 实际用电成本
- 未使用HEMS系统时的用电成本（对比参考）
- 总收益

并以结构化方式展示：
- 储能收益
- 光伏收益
- 省电收益
- 售电收益

👉 帮助用户评估系统经济价值

### 2.4 能源分项展示

通过模块化展示不同能源来源及用途：
- 光伏发电贡献
- 储能系统参与情况
- 电网交互（购电/售电）

👉 支持以金额或比例形式展示

### 2.5 数据图表区域

页面底部提供数据曲线展示（如功率曲线）：
- 支持切换不同图表类型
- 展示系统运行趋势

👉 用于分析负载变化及能源使用情况

## 3. 使用说明

1. 进入系统首页后，默认显示“今日”数据
2. 通过时间切换按钮查看不同时间段的数据
3. 观察系统拓扑图，了解当前能量流向
4. 查看右侧收益分析，评估经济表现
5. 结合底部曲线，分析系统运行趋势


:::info
- 页面数据依赖设备正常接入及通信状态
- 若数据异常或未显示，请检查设备连接及网络状态
- 不同时间维度下的数据可能存在延迟更新
:::

通过实时概览页面，用户可全面掌握能源系统运行状态，为策略优化与运行调整提供依据。
