# 策略中心

## 1. 功能概述

策略中心用于对系统运行逻辑进行配置与管理。用户可通过设置自动化策略，实现设备在不同时间、不同条件下的自动运行与调度控制。

通过策略配置，可实现：
- 按时间段控制设备运行 
- 基于条件触发自动执行动作 
- 多策略协同调度，提升系统运行效率 

<img src={require("./img/my_strategy.png").default} width="960" /> 

## 2. 页面说明

进入【策略中心】模块后，默认显示“我的策略”页面。

页面主要包括：
- **我的策略**：已创建的策略列表。页面右上角提供【**添加自动化策略**】按钮，用于创建新的策略。
- **[策略模板](#4-策略模板)**：系统预置策略
- **[电价参数](#5-电价参数)**：用于策略计算的电价设置 




## 3. 添加自动化策略

点击“**我的策略**”页面右上角【**添加自动化策略**】按钮，系统弹出策略配置窗口，用于定义策略的执行逻辑。
<img src={require("./img/add_strategy.png").default} width="480" /> 


### 3.1 基础信息

用于定义策略的基本属性：

| 参数     | 说明                                           |
| -------- | ---------------------------------------------- |
| 名称     | 自定义策略名称，便于识别                       |
| 生效日期 | - 永久<br /> - 指定日期（可设置起止时间）      |
| 生效时段 | - 全天 <br />- 指定时段（支持时间段配置）      |
| 启用状态 | 控制策略是否生效                               |
| 优先级   | 用于多策略冲突时的执行顺序（高优先级优先执行） |

当选择“指定日期”或“指定时段”时：
- 可设置起止日期范围 
- 可设置多个时间段（如分时运行） 
- 支持时间轴方式直观配置运行时间 

<img src={require("./img/strategy_period.png").default} width="600" /> 


适用于峰谷电价调度、分时控制等场景


### 3.2 条件参数

用于定义策略触发条件。

点击“+”可添加条件，例如：
- 购电价
- 售电价 

<img src={require("./img/strategy_condition.png").default} width="360" /> 

当满足设定条件时，策略将被触发执行。


### 3.3 动作参数

用于定义策略执行内容。

点击“+”可添加动作，依次完成：选择设备、选择参数、参数范围。

<img src={require("./img/strategy_action.png").default} width="360" /> 

:::warning
- 未启用的策略不会生效 
- 如果同时段下，多个策略同时满足执行条件，且控制同一个设备动作，仅执行高优先级策略。
- 条件与动作需合理匹配，否则可能导致策略无效 
- 建议避免设置冲突策略（如同时控制同一设备） 
:::


策略生效后，系统将根据设定逻辑自动执行，无需人工干预，从而实现智能化运行管理。

## 4. 策略模板

在【**策略中心**】模块中，进入“**策略模板**”页面，可查看系统提供的预置策略模板。用户可基于模板快速创建自动化策略，提高配置效率。


### 4.1 功能概述

策略模板为系统预设的典型运行策略集合，用户无需从零配置，可直接基于模板生成自动化策略，并根据实际需求进行调整。
<img src={require("./img/strategy_template.png").default} width="960" /> 

### 4.2 页面说明

页面以卡片形式展示各类策略模板，主要包括：

| 模块             | 说明                           |
| ---------------- | ------------------------------ |
| 动态电价策略模板 | 根据电价变化自动调整运行策略   |
| 自定义策略模板   | 支持用户灵活配置控制逻辑       |
| 定时充电策略模板 | 按设定时间进行充电控制         |
| 自发自用策略模板 | 优先使用本地发电，降低用电成本 |
| 需量管理策略模板 | 根据用电负荷进行优化控制       |

每个模板卡片均提供快速使用入口


### 4.3 模板加载机制

- 页面默认显示部分常用策略模板 
- 点击右上角【**刷新**】按钮后，可加载并显示更多策略模板 

若未看到完整模板列表，请先执行刷新操作


### 4.4 使用方式

策略模板可用于创建自动化策略，具体方式如下：
1. 进入“**策略模板**”页面 
2. 选择目标策略模板 
3. 点击 <img src={require("./img/add_icon.png").default} width="20" style={{verticalAlign: "middle"}}/> 进入使用 
4. 根据需要修改参数 
5. 保存为新的自动化策略 
6. 设置完成后在“**我的策略**”页面查看相关数据

<img src={require("./img/strategy_template.png").default} width="960" /> 
<img src={require("./img/add_strategy_using_template.png").default} width="960" /> 

### 4.5 与自动化策略的关系

- 自动化策略：实际执行的控制逻辑 
- 策略模板：预设方案（参考/快速创建） 

用户可通过两种方式创建策略：
- 方式一：在“我的策略”中手动创建（见[添加自动化策略](#3-添加自动化策略)） 
- 方式二：基于策略模板快速生成 

:::info
- 模板仅为参考配置，使用前建议根据实际场景调整参数 
- 不同模板适用于不同运行场景，选择时需结合实际需求 
- 模板生成策略后，可在“我的策略”中进行进一步修改 
:::

通过合理使用策略模板，用户可快速构建适用于不同场景的自动化控制策略，实现系统运行优化。

## 5. 电价参数

在【**策略中心**】模块中，进入“**电价参数**”页面，可对系统运行所需的电价信息进行配置。电价参数是自动化策略（如动态电价策略、充放电策略等）的重要输入依据。

<img src={require("./img/price_parameter.png").default} width="960" /> 


### 5.1 功能概述

本页面用于设置设备运行过程中涉及的电价参数，包括购电电价与售电电价。系统可基于电价信息，实现分时调度、峰谷套利及能量优化管理。

### 5.2 页面说明

页面主要分为两部分：

**1️⃣购电电价**

用于配置从电网购电时的电价参数。

电价右侧【**购电电价设置**】按钮进入配置页面。

电价类型选择：
- 固定电价
- 时段电价
- 动态电价

<img src={require("./img/purchase.png").default} width="360" /> 


**2️⃣ 售电电价**

用于配置向电网售电（上网）时的电价参数。

电价右侧【**售电电价设置**】按钮进入配置页面。

电价类型选择：
- 固定电价
- 动态电价

<img src={require("./img/selling.png").default} width="360" /> 

👉 用户可根据当地电价政策或用电需求进行配置。

### 5.3 使用说明

电价参数用于策略计算逻辑，如“动态电价策略”、“定时充电策略”等，均依赖电价配置。

未配置电价参数时，部分策略可能无法正常运行或效果受限


:::tip
- 请根据实际电价标准进行设置
- 分时电价需确保时间段划分准确
- 购电电价与售电电价应分别配置，避免逻辑冲突
:::

合理配置电价参数，有助于系统实现能量优化调度，提高经济收益。
