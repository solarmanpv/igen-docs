# 系统管理

## 1. 系统设置

在【**系统管理**】模块中，进入“**系统设置**”页面，可查看设备基础信息，并进行系统维护、时间配置及升级管理等操作。

<img src={require("./img/system_management.png").default} width="960" /> 

## 2. 页面说明

页面主要由三部分组成：设备信息、时间设置及系统维护操作。

### 2.1 设备配置

用于展示设备的基本信息及运行状态：
- HEMS型号（如：EMH-2）
- 序列号（SN）
- CPU占用率
- 内存占用率
- 存储空间使用情况

👉 用于判断设备当前运行状态及资源使用情况

### 2.2 时区设置

用于查看和配置系统时间信息：
- 系统时间
- 当前时区

👉 正确的时间与时区设置对于策略执行及数据记录至关重要  

页面底部提供设置入口（⚙️），可进入时区配置页面进行调整。

<img src={require("./img/time_setting.png").default} width="480" /> 


### 2.3 升级检测

用于管理系统版本及升级状态：
- 当前版本号
- 最新版本状态（如：已是最新版本）

👉 支持检测是否存在新版本

页面底部提供操作入口：
- 检查更新
- 执行升级（如有新版本）


## 3. 其他操作

页面下方提供系统维护相关功能：

**1️⃣ 系统重启**

用于重启设备系统。  
👉 适用于系统异常或配置变更后重启生效

**2️⃣ 恢复出厂设置**

将设备恢复至初始状态。

:::warning
- 所有配置（设备接入、策略等）将被清除
- 恢复后需重新配置系统
:::

**3️⃣ 配置导出**

用于导出当前系统配置文件，便于备份或迁移。

**4️⃣ 配置导入**

用于导入已有配置文件，快速恢复系统设置。

**5️⃣ 隐私政策**

用于查看系统相关的隐私政策说明。

:::tip
- 建议在进行恢复出厂或升级操作前备份配置
- 系统重启或升级过程中，请勿断电
- 时间设置错误可能导致策略执行异常
:::

通过系统管理模块，用户可对设备进行统一维护与管理，保障系统稳定运行。
